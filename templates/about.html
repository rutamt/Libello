Our Story

Summer of 2022: IB Personal Project

The personal project is a project that all IB rising sophmores are required to do over the summer. The goal of the project is to learn new skills and create something worthwhile with them. We decided to do a group project, with both of us learning different skills to fit the different jobs required for web development.

It all started with an idea.

An app that could display all of your Schoology assignments in one place.

What We Learned

Rutam

By doing this project, I’ve learned a lot about python and the coding workflow. My job was to create the system for making and managing users, which I did using a service called Auth0 I also hosted the website using Heroku, and created the flask server that Heroku is hosting. I learned a lot about apis and authentication during the project, and I used a module called Schoolopy to help with the schoology api.

Jaewon

I came into this project without much knowledge about any of the languages used for web design, so approaching this was a big learning curve. In doing this project I've gotten the hang of languages like CSS, HTML, Javascript, and libraries like Boostrap 5 or jQuery. I've also been able to try functionalities like CSS keyframes and animations as well as Javascript modules. Overall, this is a good start to my web development journey.
